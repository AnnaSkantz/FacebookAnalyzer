## Installation

1. Install [node](https://nodejs.org/)
2. Clone this repo
3. Navigate to `<this repo>/socialauth/`
4. Run `npm install -g http-server`
5. Run `http-server ./ -p 80`
6. Go to localhost:80 on your preferred web browser
